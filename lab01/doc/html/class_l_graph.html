Detailed Description

template<typename NT, typename ET>
class LGraph< NT, ET >

Adjacent list graph class.

LGraph implemented using lists of nodes adjacent to each node. Consumes O(|N| + |E|) space, where |N| - number of nodes, |E| - number of edges.
